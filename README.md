# maintenance-qr-tool
Small application to generate QR codes which will make getting information easier and more efficient

Download all the files and be sure to install whats under "Requires: " this can be done by running:

'''bash
pip install flask pillow qrcode
'''
I use VSC to run it so in terminal I use:

'''bash
python app.py
'''
I then open a browser and navigate to:
127.0.0.1:5000

Requires:
flask
qrcode
pillow
